Anatomy of a Web Page

  • HTML is for Content
  • CSS is for Presentation
  • JavaScript is for Interactivity

HTML

  • HyperText Markup Language
  • Define and describe content: links, headers, footers, paragraphs, images, etc.

CSS

  • Cascading Style Sheets
  • Describing the look and formatting of a document written in a markup language(HTML, SVG, VML, MathML, etc.)

JavaScript

  • Add interactivity and behavior to HTML
  • Make content dynamic

The Nature of JavaScript

  • It’s dynamic
  • It’s dynamically typed
  • It’s functional and object-oriented
  • It fails silently
  • It’s deployed as source code
  • It’s part of the web platform

Elegant Parts of JavaScript

  • First-class functions
  • Closures
  • Prototypes
  • Object literals: {}
  • Array literals: []

Programming languages that influenced JavaScript

  • Java is the role model for JS's syntax, data type and memory management
  • AWK inspired JS's functions
  • Scheme is the reason that JS has first-class functions
  • Self is responsible for JS’s prototype inheritance
  • Perl and Python influenced JS’s handling of strings, arrays, and regular expressions
  • HyperTalk influenced how JS was integrated into web browser, such as `onclick`
